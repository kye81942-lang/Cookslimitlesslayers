# Cook's Limitless Layers

A modern, fully responsive website for Cook's Limitless Layers - a creative 3D printing and custom products business.

## Features

✨ **Modern Design**
- Sleek dark theme with cyan accents
- Fully responsive design (mobile, tablet, desktop)
- Smooth animations and transitions
- Sticky navigation header

📦 **Product Showcase**
- 4 main product categories:
  - 3D Prints
  - Apparel
  - Household Accessories
  - Everyday Essentials
- Interactive product cards with hover effects

🛒 **E-Commerce Ready**
- Shop Now call-to-action button
- Explore buttons for each category
- Product grid layout

📝 **Custom Orders**
- Contact form for custom project requests
- Form validation and submission handling

📞 **Contact Section**
- Complete business contact information
- Easy-to-find location details

## Getting Started

1. Clone this repository:
```bash
git clone https://github.com/kye81942-lang/CooksLimitlessLayers.git
```

2. Open `index.html` in your browser

3. For local development with a server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js http-server
npx http-server
```

Then visit `http://localhost:8000`

## Customization

### Update Business Information
Edit the contact section in `index.html`:
- Email: info@cookslimitlesslayers.com
- Phone: (555) 123-4567
- Location: Your City, Your State

### Modify Colors
Primary colors in `style.css`:
- Cyan Accent: `#00d9ff`
- Dark Background: `#050505`
- Card Background: `#1a1a1a`

### Add Product Images
Place images in the `images/` folder and update the `.product-image` divs in `index.html`
